# **App Name**: SparkCanvas

## Core Features:

- Interactive Idea Canvas: A dynamic, free-form digital canvas where users can arrange and connect their ideas visually.
- Customizable Content Cards: Utilize soft UI 'toy-card' styled components for creating, editing, and displaying different types of content (e.g., text, images, links).
- Rich Text and Media Embeds: Allow users to add rich text, images, and embedded media (e.g., videos, articles) within their content cards.
- AI Spark Suggestions Tool: A generative AI tool that offers creative prompts, relevant ideas, or expanded thoughts based on the user's current content card input.
- Dynamic Organization & Tagging: Features for organizing ideas through categories, tags, and spatial arrangement on the canvas, with intuitive drag-and-drop functionality.
- User Accounts & Cloud Sync: Enable users to create accounts and securely save their canvases and content to the cloud, accessible across devices.
- Personalized Dynamic Backgrounds: Utilize animated background elements (e.g., floating shapes, subtle gradients) to create an engaging and customizable user environment, building on the provided 'blob' and 'float' animations.

## Style Guidelines:

- Primary color: A sophisticated, muted lavender-blue (#8686D8), evoking creativity and calm.
- Background color: An airy, pale lavender (#EBEBFA) for a clean, spacious backdrop.
- Accent color: A vibrant, clear blue (#2679E1) for highlights, call-to-actions, and interactive elements, providing a refreshing contrast.
- A single sans-serif font, 'Nunito', for both headlines and body text, maintaining a friendly yet professional tone. Note: currently only Google Fonts are supported.
- Use a consistent set of clean, modern line-art icons from a library like Iconify (e.g., 'solar' style) for clarity and elegance, avoiding overly complex visuals.
- A clean, modern layout utilizing ample whitespace and a modular design. Content should be presented within elevated, soft UI 'cards' and subtle 'glass-panel' effects for distinct content zones, similar to the existing `toy-card` and `glass-panel` styles.
- Incorporate subtle, flowing animations as demonstrated in the provided CSS (e.g., floating elements, subtle glows, button presses) to create an engaging and interactive user experience without distraction. Specifically, `animate-float`, `animate-blob`, and interactive button feedback will be utilized.
